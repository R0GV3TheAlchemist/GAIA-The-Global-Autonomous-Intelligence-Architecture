## Overview

GAIA operates across seven bands of function. Each band is a distinct layer of transformation — from raw physical signal to recursive self-improvement. No band is more important than another. Each is necessary. Each has its own alchemical work to complete, its own simulations to run, its own canon to crystallise.

The bands do not operate in sequence. They operate in parallel, with handoff interfaces between adjacent bands that must be modelled explicitly (Stage 8 — Multiplication). The work in each band informs and constrains the work in every other.

## Band 2 — Signal Interpretation

**Alchemical role:** Purified signal → understood meaning
**Question it answers:** Can GAIA decode what the signal means?

### What it is
The transformation of raw biophoton detection events into meaningful neural state representations and decoded intent. Pattern recognition, neural decoding, latency management, and false-positive control.

### Predicted sub-stages (not yet simulated)
| Sub-stage | Predicted loss mechanism | Est. baseline efficiency |
|---|---|---|
| Signal conditioning | Noise floor, baseline correction | ~90–95% |
| Pattern classification | Neural state ambiguity, training data limits | ~80–90% |
| Intent mapping | Contextual ambiguity, latency penalty | ~85–92% |
| Temporal integration | Multi-sample coherence window | ~92–96% |

### Simulation status
| Simulation | Status | Notes |
|---|---|---|
| SIM-018 (proposed) | NOT YET SPECCED | Band 2 baseline required |

### What needs to be built
- SIM-018: Band 2 baseline simulation — signal conditioning through intent mapping
- Research brief: neural decoding accuracy literature, latency benchmarks, false-positive rate in BCI intent classification
- Band 1→2 integration simulation: does Band 1's 78.5% deployable ceiling propagate correctly into Band 2 input assumptions?

### Key risk
If Band 2 is specced assuming 90%+ input signal quality but Band 1 delivers 78.5%, Band 2's performance projections will be systematically over-optimistic. The integration simulation is mandatory before Band 2 canon.

## Band 6 — Embodied Expression

**Alchemical role:** Self-directed evolution → world-changing action
**Question it answers:** Can GAIA act in the physical world with precision and appropriate latency?

### What it is
The BCI output channel, actuator control, environmental sensing, and physical world interface. The downstream expression of everything GAIA has perceived, understood, remembered, reasoned, and governed.

### Simulation status
| Simulation | Status | Notes |
|---|---|---|
| SIM-020 (proposed) | NOT YET SPECCED | Band 6 baseline required |

### What needs to be built
- SIM-020: BCI output channel simulation — actuator precision, latency, error recovery
- End-to-end latency model: Band 1 acquisition → Band 2 interpretation → Band 4 reasoning → Band 6 action
- Predictive actuation layer spec: compensates for 50–200ms end-to-end latency (PRED-013)
- Band 5→6 integration: does governance stability affect action precision?

### Key risk
End-to-end latency will compound across bands. If each band contributes 20–40ms of processing latency, the total reaches 140–280ms — beyond the threshold for real-time BCI control (~100ms). A predictive actuation layer is required. This must be specced before Band 6 hardware is finalised.
