## SIM-017 Interaction — Critical Note

SIM-017 Pass 1 confirmed the structural connectivity floor mechanism: high-connectivity nodes in the memory architecture resist decay independent of access frequency. The KG in SIM-006 uses confidence scores and provenance fields as analogues of connectivity weighting.

**Implication:** The 50-cycle gardening cadence was set before the connectivity floor finding. If confidence score decay in the KG follows a similar floor mechanism, the gardening cadence may be overly conservative (could be tightened). Conversely, if KG edges lack the structural redundancy of the memory Relational Index, the cadence may need to be more frequent.

**Required action before SIM-006 next pass:** A joint research brief covering both SIM-006 and SIM-017 Band 3→4 interaction. This brief must answer: does the KG confidence score decay model replicate the structural connectivity floor, or does it need an explicit floor mechanism added?
