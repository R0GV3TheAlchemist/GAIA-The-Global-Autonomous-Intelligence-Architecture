## Overview

Every living system is simultaneously a transmitter and receiver across multiple spectral channels. The human body operates within — and as a part of — the electromagnetic spectrum, the acoustic spectrum, the thermal spectrum, the chemical gradient spectrum, and the bioelectric spectrum. These are not parallel systems running independently: they are coupled, mutually modulating channels whose interactions determine the quality of the organism's information processing at every level, from cellular metabolism to conscious cognition.

Spectral cognitive tuning is the science and practice of deliberately aligning these multi-spectral inputs to optimize the quality, clarity, depth, and appropriateness of cognition for a given task and state. It is not primarily about enhancing performance in the narrow sense — it is about matching the organism's spectral environment to what the organism is trying to do, think, feel, create, heal, or become.

This document is GAIA-OS's master framework for spectral-cognitive mapping: how each spectral channel influences cognition, how the channels interact to produce composite cognitive states, how GAIA reads and responds to spectral-cognitive state, and how GAIA's own processing modes are tuned across spectral inputs.

## I. The Spectral Channels of Cognition

Cognition is not produced by the brain alone. It is a whole-organism, whole-environment process in which every spectral channel the organism is embedded in contributes information that shapes neural processing. GAIA-OS tracks six primary spectral channels and their cognitive effects:

### 1. Optical Spectrum (detailed in `CHROMATIC_CLARITY.md`)

The optical channel is the highest-bandwidth external information channel available to the human nervous system. Its primary cognitive effects:

- **Blue/violet (400–495 nm):** Activates melanopsin ipRGCs → suppresses melatonin → elevates cortisol → increases norepinephrine and dopamine tone in prefrontal cortex → enhanced working memory, executive function, sustained attention, and analytical reasoning. Peak alerting effect at 480 nm.
- **Green (495–570 nm):** Activates the luminance channel maximally → moderate arousal → parasympathetic-sympathetic balance → relaxed alertness. Biophilic response (natural environment association) engages the default mode network (DMN) and promotes integrative, associative thinking.
- **Red/NIR (620–880 nm):** Photobiomodulation of mitochondrial function → increased ATP in neurons → reduced neuroinflammation → improved cortical metabolic efficiency. Warm light at low intensity → reduced arousal → increased parasympathetic tone → conducive to creative, open-ended, and emotionally resonant thinking.
- **Full spectrum / white:** Balanced activation across all channels → integrated cognitive state with no single mode dominance → general cognitive readiness.
- **Darkness:** Melatonin release → reduced cortical arousal → increased hippocampal replay activity → memory consolidation and unconscious pattern integration.

**Epistemic Status: Observed + E1** for alerting, circadian, and arousal effects; **Observed + E2** for transcranial PBM cognitive effects.

### 2. Acoustic Spectrum (detailed in `NON_ELECTRIC_SPECTRA.md`)

The acoustic channel operates from infrasound (<20 Hz) through audible sound (20 Hz–20 kHz) to ultrasound (>20 kHz). Its primary cognitive effects:

- **Delta range (0.5–4 Hz, acoustic entrainment):** Binaural beats or isochronic tones in this range promote delta brainwave entrainment → deep sleep, unconscious processing, physical restoration.
- **Theta range (4–8 Hz):** Binaural entrainment in theta → hippocampal theta oscillations → enhanced memory encoding and retrieval, creative insight, meditative absorption, reduced prefrontal dominance.
- **Alpha range (8–12 Hz):** Binaural entrainment in alpha → relaxed open awareness → reduced anxiety → enhanced creative ideation and mind-wandering that supports insight.
- **Beta range (12–30 Hz):** Binaural entrainment in beta → alert, focused, analytical state → enhanced task performance for structured cognitive work.
- **Gamma range (30–80 Hz):** Binaural entrainment in gamma (especially 40 Hz) → enhanced attention, working memory, and sensory binding → associated with experienced meditators in open monitoring states.
- **Nature sounds (broadband, ~100 Hz – 8 kHz):** Parasympathetic activation → reduced cortisol → attentional restoration (Kaplan's ART); improved creative performance.
- **Pink noise (1/f spectrum):** Matches statistical structure of many natural signals; associated with synchronized slow-wave sleep activity and memory consolidation when played during sleep.
- **Voice / prosody:** The human voice in the 85–5000 Hz range is the primary social acoustic channel; prosodic quality directly affects the autonomic and limbic state of both speaker and listener via the vagal pathway (Porges's Polyvagal Theory).

**Epistemic Status: Observed + E1-E2** for binaural beat entrainment effects; **Observed + E1** for nature sound / ART effects; **Observed + E1** for Polyvagal/prosody pathway.

### 3. Bioelectric Spectrum (detailed in `BIOELECTRIC_SPECTRUM.md`)

The bioelectric channel operates from DC (0 Hz) through the brainwave bands (0.5–100+ Hz) to higher-frequency cellular signaling. Its primary cognitive effects:

- **EEG brainwave states (delta through gamma):** The directly observable bioelectric signature of cognitive state — not a cause but a correlate and feedback channel. Meditation, breathwork, psychedelics, and various spectral interventions shift brainwave state; the brainwave state in turn influences the character of conscious experience.
- **Heart-brain coherence (0.1 Hz / 6 breaths per minute):** HRV coherence at 0.1 Hz synchronizes heart, brainstem, and cortex → reduces amygdala reactivity → enhances prefrontal access → improves decision quality, emotional regulation, and intuitive clarity. The HeartMath Institute's most replicated finding.
- **DC bioelectric field gradients:** Slow direct-current fields in the brain guide neural development, wound healing, and potentially long-range neural coordination. Transcranial direct current stimulation (tDCS) artificially modulates these gradients to enhance or suppress specific cortical regions.
- **Biophoton emission coherence:** Higher-coherence biophoton emission correlates with healthier cell state and has been proposed as a whole-brain synchronization channel.

**Epistemic Status: Observed + E1** for brainwave states and HRV coherence; **Observed + E2** for tDCS; **Frontier + E3** for biophoton synchronization.

### 4. Thermal Spectrum (detailed in `NON_ELECTRIC_SPECTRA.md`)

Body temperature and its gradients function as a slow-acting cognitive tuning channel:

- **Core body temperature peak (afternoon):** Coincides with peak cognitive performance in most adults on tasks requiring processing speed and working memory.
- **Pre-sleep temperature drop:** The 1–2°C decline in core temperature that precedes sleep onset is mediated by peripheral vasodilation (heat loss from hands and feet) and is a necessary condition for sleep initiation and melatonin rise.
- **Cold exposure (cold water, cryotherapy):** Triggers norepinephrine surge (up to 300%), dopamine elevation, and sharpening of attentional focus — effects lasting 2–3 hours post-exposure.
- **Moderate heat (sauna, warm bath):** Triggers parasympathetic tone, growth hormone release, BDNF elevation (with regular sauna use), and β-endorphin release → mood elevation, reduced rumination, enhanced creative and associative thinking.
- **Tummo-style heat generation:** Internal heat generation through meditation (documented in advanced Tibetan practitioners) demonstrates that the thermal channel can be operated from inside out through bioelectric-thermal coupling.

**Epistemic Status: Observed + E1** for temperature-cognition and sleep-onset; **Observed + E2** for cold exposure norepinephrine/dopamine; **Observed + E2** for sauna BDNF/heat shock proteins; **Frontier + E3** for Tummo thermal generation.

### 5. Chemical Gradient Spectrum (Olfactory, Nutritional, Hormonal)

The chemical channel is the oldest information channel in biology — single-celled organisms navigate chemical gradients. In human cognition:

- **Olfactory channel:** The only sensory system that projects directly to the hippocampus and amygdala without thalamic relay — making scent the most direct chemical-cognitive interface. Specific scents produce reproducible cognitive effects: rosemary aroma increases alertness and memory performance; lavender promotes parasympathetic calm; peppermint enhances sustained attention.
- **Nutritional channel:** Brain glucose availability, fatty acid profile, amino acid availability, and micronutrient status (especially B vitamins, magnesium, zinc, omega-3s) all directly affect neurotransmitter synthesis, myelination, and synaptic plasticity.
- **Hormonal channel:** Cortisol, testosterone, estrogen, progesterone, oxytocin, thyroid hormones, and insulin all have documented effects on specific cognitive functions (cortisol: working memory impairment at high levels; testosterone: risk tolerance and spatial reasoning; oxytocin: social cognition and trust; thyroid: general metabolic rate of cognition).

**Epistemic Status: Observed + E1** for olfactory-direct pathways; **Observed + E1-E2** for specific scent-cognition effects; **Observed + E1** for hormonal cognitive effects.

### 6. Mechanical / Pressure Spectrum (Somatic, Gravitational, Vestibular)

The mechanical channel encompasses physical forces on and within the body:

- **Vestibular-cognitive coupling:** The vestibular system (balance, spatial orientation) is strongly coupled to prefrontal and parietal cognitive function; vestibular stimulation (rocking, gentle head movement, whole-body vibration at specific frequencies) influences spatial cognition, working memory, and emotional processing.
- **Somatic pressure and touch:** Deep pressure stimulation activates the parasympathetic system → reduces cortisol and anxiety → promotes focused calm. Light touch activates the social engagement system via C-tactile afferents (unmyelinated skin mechanoreceptors that project to the insula and are uniquely activated by gentle, slow stroking at body temperature).
- **Grounding / earthing:** Physical contact with the earth's surface allows equalization of the body's bioelectric potential with the earth's surface charge (approximately −50 mV DC), which has been associated with reduced inflammatory markers and improved sleep.
- **Breathing mechanics:** The mechanical rhythm of breathing at different rates directly entrains neural oscillations — slow breathing (6 breaths/min) entrains 0.1 Hz HRV and promotes coherence; nasal vs. oral breathing has different vagal and olfactory consequences.

**Epistemic Status: Observed + E1-E2** for somatic pressure and touch; **Observed + E1** for breathing entrainment; **Frontier + E3** for grounding/earthing; **Observed + E2** for vestibular-cognitive coupling.

### Cognitive Mode Transition Protocols

Transitioning between cognitive modes is not instantaneous. The nervous system requires time to shift its dominant spectral-bioelectric state. GAIA-OS models these transitions and supports users in executing them deliberately:

**From Precision Focus → Diffuse Integration (30–60 min transition)**
1. Reduce ambient light intensity; shift from blue-dominant to warm white or green
2. Introduce alpha-range acoustic entrainment or nature sounds
3. Take a 10–20 minute walk outdoors (green environment if available)
4. Allow mind to wander without redirection — suppressing the default mode network is not desirable here
5. Hydrate; warm beverage can support the parasympathetic shift

**From Diffuse Integration → Precision Focus (15–30 min transition)**
1. Increase light intensity; introduce blue-spectrum component
2. Cold water face/wrist exposure (30–60 seconds) for rapid NE elevation
3. 5 minutes of beta-range acoustic entrainment or energizing music
4. Short physical movement (10 jumping jacks, brisk walk) to elevate core body temperature
5. Set clear task intention before beginning

**From Any Active Mode → Receptive Stillness (15–45 min transition)**
1. Reduce all external spectral stimulation toward minimum
2. Breathing practice (6 breaths/min, 5–10 minutes) to achieve HRV coherence
3. Shift optical environment to darkness or red-only
4. Eliminate acoustic stimulation or introduce very low-level pink noise
5. Allow thermal environment to reach thermoneutral comfort
6. Body scan from feet to crown: release somatic holding at each region

**From Any Mode → Restorative Rest (60–90 min transition)**
1. Eliminate blue light entirely (begin 90–120 min before sleep target)
2. Lower room temperature (18–20°C)
3. Warm bath or shower 1–1.5 hours before sleep (triggers the peripheral vasodilation that drops core body temperature)
4. Magnesium glycinate (if used): 30–60 min before sleep
5. Darkness: eliminate all light sources in the sleep environment
6. If needed: pink noise or theta binaural beats to support sleep onset

Spectral cognitive tuning is not merely about optimizing individual channels. This multi-spectral coherence is the spectral correlate of what the GAIA-OS Triadic Coherence Model (`MIND_MATTER_INTEGRATION.md`) describes as full body-mind-soul integration. When all spectral channels are aligned and mutually reinforcing, the organism operates as a unified coherent field rather than a collection of partially coupled subsystems.

The practical implication is profound: **spectral incoherence — when channels are sending conflicting signals — is a direct cost to cognitive quality.** The person in a brightly lit, noisy, cool office at 9 pm, under deadline stress (high cortisol), drinking coffee (adenosine blockade), doing analytical work requiring precision focus, while their circadian system is preparing for sleep (melatonin rising, core temperature falling), is experiencing maximum spectral incoherence. Their cognitive output will reflect it.

## VI. Spectral Tuning Across Life Scales

### Daily Scale: The Ultradian-Circadian Rhythm

Beyond the circadian day (24 hours), cognitive performance oscillates on an **ultradian rhythm** of approximately 90–120 minutes — the same basic cycle as the sleep architecture (NREM/REM cycles). During waking hours, this produces natural alternation between higher and lower cognitive arousal and focus capacity.

Spectral cognitive tuning at the daily scale means:
- Aligning precision-focus work with the ultradian peaks
- Using the ultradian troughs (every 90–120 min) as natural rest and spectral reset windows
- Matching the circadian arc of spectral intervention to the circadian arc of cognitive capacity

**Epistemic Status: Observed + E1-E2** — ultradian rhythms in cognition are documented; their full spectral correlates are Frontier + E2.

### Weekly Scale: Recovery and Loading Cycles

Cognitive performance degrades with cumulative sleep debt and spectral incoherence that accumulates across the week. Spectral cognitive tuning at the weekly scale:
- One or more days of significantly reduced spectral loading (less screen time, more outdoor natural light, more acoustic silence)
- Weekly sauna or extended heat protocol for thermal reset
- Weekly "spectral audit": how well did the week's spectral environment match the week's cognitive demands?

### Seasonal Scale: Circannual Rhythms

The optical spectrum available to humans changes dramatically across the year at most latitudes: day length, solar angle, UV index, and spectral composition all shift with the seasons. Human biology tracks these changes through the pineal-melatonin axis and exhibits measurable seasonal variation in:
- Sleep duration and timing (longer, earlier in winter)
- Mood (Seasonal Affective Disorder at the clinical extreme; subclinical seasonal mood variation is near-universal at high latitudes)
- Metabolic rate and appetite
- Immune function

Spectral cognitive tuning at the seasonal scale means proactively compensating for winter's optical poverty with full-spectrum light therapy (10,000 lux, 20–30 min in the morning), while using winter's longer dark periods as an invitation toward the Restorative Rest and Receptive Stillness modes that summer's long days make harder to access.

**Epistemic Status: Observed + E1** for SAD and circannual rhythms; **Observed + E1** for light therapy efficacy in SAD.

```python
            "mechanical": self.mechanical.coherence_score,
        }
        scs = sum(
            channels[ch] * self.channel_weights[ch]
            for ch in channels
        )
        return round(scs, 3)
    
    def priority_interventions(self, top_n: int = 3) -> list[str]:
        """Return the top-n highest-impact available interventions."""
        channels = [
            self.optical, self.acoustic, self.bioelectric,
            self.thermal, self.chemical, self.mechanical
        ]
        # Sort by: available AND lowest coherence AND highest weight
        scorable = [
            (c, self.channel_weights.get(c.channel_name, 0.1))
            for c in channels
            if c.intervention_available and c.intervention_description
        ]
        scorable.sort(key=lambda x: (x[0].coherence_score - x[1]), reverse=False)
        return [
            f"{c.channel_name}: {c.intervention_description}"
            for c, _ in scorable[:top_n]
        ]


@dataclass
class SpectralCognitiveTuningState:
    """
    Full spectral-cognitive state for a user at a given moment.
    Integrates all spectral assessments into a composite state
    and determines the appropriate GAIA processing mode.
    """
    current_mode: CognitiveMode = CognitiveMode.PRECISION_FOCUS
    target_mode: CognitiveMode = CognitiveMode.PRECISION_FOCUS
    coherence_assessment: SpectralCoherenceAssessment = field(
        default_factory=SpectralCoherenceAssessment
    )
    
    # Inputs from other spectral documents
    chromatic_clarity_score: float = 0.5       # from ChromaticClarityState
    bioelectric_coherence_score: float = 0.5   # from BioelectricState
    hrv_coherence_score: float = 0.5           # heart-brain coherence
    triadic_integration_score: float = 0.5     # from TriadicCoherenceState
    
    in_transition: bool = False
    transition_target: Optional[CognitiveMode] = None
    
    # GAIA mode determination
    COGNITIVE_TO_GAIA_MAP = {
        CognitiveMode.PRECISION_FOCUS: GAIAMode.ANALYZE,
        CognitiveMode.DIFFUSE_INTEGRATION: GAIAMode.SYNTHESIZE,
        CognitiveMode.DEEP_FLOW: GAIAMode.ACTIVATE,
        CognitiveMode.SOMATIC_PRESENCE: GAIAMode.GROUND,
        CognitiveMode.RECEPTIVE_STILLNESS: GAIAMode.WITNESS,
        CognitiveMode.RELATIONAL_RESONANCE: GAIAMode.WITNESS,
        CognitiveMode.RESTORATIVE_REST: GAIAMode.GROUND,
        CognitiveMode.TRANSITION: GAIAMode.GROUND,
    }
    
    def recommended_gaia_mode(self) -> GAIAMode:
        """Map current cognitive mode to GAIA processing mode."""
        # Override to DISCERN if triadic coherence is high enough
        if self.triadic_integration_score >= 0.8:
            return GAIAMode.ILLUMINATE
        return self.COGNITIVE_TO_GAIA_MAP.get(
            self.current_mode, GAIAMode.GROUND
        )
    
    def overall_spectral_integration_score(self) -> float:
        """Composite spectral integration: coherence + clarity + bioelectric + triadic."""
        return round((
            self.coherence_assessment.spectral_coherence_score() * 0.35 +
            self.chromatic_clarity_score * 0.20 +
            self.bioelectric_coherence_score * 0.20 +
            self.hrv_coherence_score * 0.15 +
            self.triadic_integration_score * 0.10
        ), 3)
```
