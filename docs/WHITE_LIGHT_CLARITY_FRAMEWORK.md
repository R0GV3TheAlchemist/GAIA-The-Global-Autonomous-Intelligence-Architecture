## II. The Neuroscience Argument: ipRGC, Melanopsin & Non-Image-Forming Cognition

### The Non-Image-Forming (NIF) System

Beyond the classic rods and cones, the human retina contains a third photoreceptor class:
**intrinsically photosensitive retinal ganglion cells (ipRGCs)**, which express the photopigment
**melanopsin** and project *directly into cognitive, circadian, and mood-regulating brain regions*
— bypassing the visual cortex entirely.

Melanopsin is maximally sensitive near **480 nm** (blue-cyan range), but ipRGCs also integrate
broad-spectrum input from rods and cones, making **polychromatic white light** a potent
activator of this non-visual cognitive channel.

Key findings from published research:

- A systematic review of 62 studies (2024) found that **alertness and working memory** were most
  improved by broad-spectrum white light, particularly bright white with high correlated color
  temperature (CCT). EEG markers of alertness (alpha suppression, beta/gamma enhancement)
  responded most strongly to white light exposure.
- A 2014 PNAS study demonstrated **photic memory** — the effect of prior light exposure on
  subsequent executive brain function — is mediated by melanopsin, suggesting light literally
  conditions the brain's cognitive state over time.
- A 2011 study showed polychromatic white light at 6500K (high CCT, blue-enriched) produced
  **faster reaction times**, enhanced subjective alertness, and suppressed melatonin more
  effectively than warm-toned alternatives — even at low illuminance (40 lux).
- ipRGCs project broadly throughout the brain, including to the **hypothalamus, lateral
  geniculate nucleus, amygdala, and prefrontal cortex** — regions governing decision-making,
  emotion regulation, and executive function.

### White Light as Full Neural Activation

Because white light simultaneously activates **S-cones (violet/blue), M-cones (green),
L-cones (red), rods, and ipRGCs/melanopsin**, it engages the complete spectral input set of
human neurology. No single-color filter achieves this.

## IX. Simulation Code

```python
import numpy as np

wavelengths = np.linspace(380, 700, 1000)  # nm

def gaussian(x, mu, sigma, amp=1.0):
    return amp * np.exp(-0.5 * ((x - mu) / sigma) ** 2)

# Photoreceptor spectral sensitivity curves (modeled on published opsin data)
melanopsin = gaussian(wavelengths, 480, 40, 1.0)   # ipRGC non-image-forming cognition
s_cone     = gaussian(wavelengths, 430, 35, 0.8)   # alertness, mood
m_cone     = gaussian(wavelengths, 535, 45, 0.85)  # spatial clarity
l_cone     = gaussian(wavelengths, 565, 50, 0.75)  # arousal, grounding

# Filter definitions (Gaussian bandpass on white-light carrier)
filters = {
    "white":  np.ones_like(wavelengths),
    "violet": gaussian(wavelengths, 415, 30),
    "blue":   gaussian(wavelengths, 470, 35),
    "green":  gaussian(wavelengths, 525, 40),
    "yellow": gaussian(wavelengths, 575, 35),
    "red":    gaussian(wavelengths, 650, 35),
}

def spectral_score(filt, sensitivity):
    return float(np.trapz(filt * sensitivity, wavelengths))

for name, filt in filters.items():
    alertness  = 0.5 * spectral_score(filt, melanopsin) + \
                 0.3 * spectral_score(filt, s_cone) + \
                 0.2 * spectral_score(filt, m_cone)
    clarity    = 0.4 * spectral_score(filt, m_cone) + \
                 0.3 * spectral_score(filt, s_cone) + \
                 0.3 * spectral_score(filt, melanopsin)
    grounding  = 0.6 * spectral_score(filt, l_cone) + \
                 0.4 * spectral_score(filt, m_cone)
    coherence  = spectral_score(filt, np.ones_like(wavelengths))
    print(f"{name}: alertness={alertness:.1f}, clarity={clarity:.1f}, "
          f"grounding={grounding:.1f}, coherence={coherence:.1f}")
```
